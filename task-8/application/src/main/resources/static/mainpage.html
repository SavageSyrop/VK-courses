This server has one REST controller with GET and POST requests

GET: localhost:3466/products
Use to watch information about products. Is open for guests and managers.

POST: localhost:3466/products
Adds product to bd, if organisation is not found creates it. Open for managers only. Use request body to send JSON

POST: localhost:3466/products/delete?productName=product
Deletes all products with name equal to query param from request. Open for managers only.

GET: localhost:3466/products/{companyName}
Use to watch information about all products from chosen company. Is open for guests and managers.
